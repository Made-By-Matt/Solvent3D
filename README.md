# Solvent3D

**Solvent** is a QOL Modded client of G3D built off of droplets utils

## What does it add?

- Mod Support
- Keybinds
- Functions
- VS Support
- Slight Optimization (Hey a frame or two is nice!!)
- ...And ofc more to come...

## Things to note

Solvent3D is a modded client built off of droplets utils 2.2 with new aditions and a mod editor

> [!CAUTION]
> Solvent3D has a built-in updater mod when the updater is ran it will delete you project SO BE CAREFUL!!!

## Getting Started

The only thing you need to begin using Glue3D is the .sb3 file provided in this repository. Simply download it by selecting "Code" and "Download ZIP."

Now that you have your copy, it's time to read the documentation :).
